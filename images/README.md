# Mag Electric — Images

Drop your photos into this folder. Recommended file names for use on the website:

## Logo
- `logo.png` — Main circular badge logo (transparent background, PNG)
- `logo-white.png` — White version if available

## Hero / Banner
- `hero-bg.jpg` — Hero background (truck, job site, or team photo — landscape, 1920×1080 min)
- `truck.jpg` — The wrapped truck (great for hero or about section)

## Work Photos (gallery section — add as many as you want)
- `work-panel-01.jpg` — Panel upgrade work
- `work-panel-02.jpg` — Another panel photo
- `work-ev-charger-01.jpg` — EV charger install
- `work-span-01.jpg` — SPAN smart panel install
- `work-lighting-01.jpg` — Lighting / recessed lights
- `work-rewire-01.jpg` — Rewire project
- `work-remodel-01.jpg` — Kitchen or bath remodel wiring
- `work-outdoor-01.jpg` — Outdoor / exterior lighting

## Team
- `matthew.jpg` — Photo of Matthew Gil (for about/why-us section)

## Notes
- Use JPG for photos (keep under 500KB each — compress at squoosh.app)
- Use PNG only for logos/graphics with transparency
- Landscape photos work best for hero/banner areas
- Square or portrait photos work for the gallery grid
